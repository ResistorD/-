# Оптимизация кода под эскизы Figma

## Обзор

Этот проект теперь включает в себя набор инструментов и рекомендаций для оптимизации кода под эскизы Figma. Мы внедрили систему компонентов и принципов, которые помогают достичь точного соответствия между дизайном в Figma и финальным пользовательским интерфейсом.

## Структура файлов

- `/lib/figma/figma_guidelines.dart` - основные компоненты и системы для Figma-оптимизации
- `/lib/figma/figma_integration_example.dart` - пример использования компонентов
- `/FIGMA_OPTIMIZATION_GUIDE.md` - подробное руководство по оптимизации

## Компоненты Figma-оптимизации

### FigmaDesignSystem
Класс, содержащий стандартные значения для:
- Системы отступов (baseSpacingUnit)
- Радиусов скруглений
- Типографской шкалы

### FigmaSizedBox
Виджет для создания отступов на основе Figma-единиц измерения.

### FigmaContainer
Контейнер с предустановленными стилями, соответствующими Figma-дизайну.

## Пример использования

```dart
// Использование Figma-контейнера
FigmaContainer.card(
  child: Column(
    children: [
      Text('Заголовок', style: FigmaDesignSystem.typography['titleLarge']),
      FigmaSizedBox.unit(heightFactor: 1), // 8dp отступа
      Text('Содержание', style: FigmaDesignSystem.typography['bodyMedium']),
    ],
  ),
),
```

## Интеграция с существующим кодом

Мы уже интегрировали Figma-компоненты в существующий виджет `EntryCard` для демонстрации подхода. Все новые компоненты могут использовать систему типографики и отступов из FigmaDesignSystem.

## Рекомендации

Для получения полной информации об оптимизации кода под эскизы Figma, пожалуйста, ознакомьтесь с полным руководством в файле `FIGMA_OPTIMIZATION_GUIDE.md`.

## Результат

Внедрение этих компонентов и принципов позволяет:
- Достигнуть большего соответствия между дизайном в Figma и финальным интерфейсом
- Упростить процесс верстки новых экранов
- Поддерживать консистентность визуального оформления по всему приложению
- Ускорить разработку за счет готовых компонентов
