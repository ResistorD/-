# Использование эскизов Figma для приложения Pressure Diary

## Обзор

Этот документ описывает, как использовать эскизы Figma для приведения приложения Pressure Diary к задуманному дизайну. Эскизы содержат визуальные макеты интерфейса, которые должны быть реализованы в коде приложения.

## Структура папки

- `main-design.fig`: Основной файл дизайна Figma
- `components.fig`: Компоненты интерфейса
- `screens.fig`: Макеты отдельных экранов приложения
- `style-guide.fig`: Гайды по стилю, цветам, типографике

## Как использовать эскизы

### 1. Анализ макетов

- Изучите каждый экран в файлах Figma
- Определите основные компоненты интерфейса
- Обратите внимание на цветовую палитру и шрифты
- Заметьте взаимодействия между элементами

### 2. Адаптация приложения

#### Цветовая схема
- Используйте цвета из эскизов для темы приложения
- Обновите файл `/lib/theme/` в соответствии с гайдами

#### Компоненты
- Реализуйте компоненты, как показано в эскизах
- Обновите файлы в `/lib/widgets/` для соответствия дизайну

#### Экраны
- Сравните существующие экраны с макетами в Figma
- Обновите файлы в `/lib/screens/` для соответствия дизайну

### 3. Типографика

- Используйте шрифты и размеры текста из эскизов
- Обновите стили в теме приложения

### 4. Отступы и размеры

- Следуйте указанным размерам элементов
- Используйте систему отступов из эскизов
- Обратите внимание на расстояния между элементами

## Интеграция с кодом

1. Извлеките цвета из эскизов и определите их в константах
2. Создайте виджеты, соответствующие компонентам из Figma
3. Обновите макеты экранов в соответствии с макетами
4. Протестируйте соответствие на разных размерах экранов

## Ресурсы

- Иконки: экспортируйте из Figma в формате SVG или PNG
- Изображения: используйте качество, подходящее для мобильных устройств
- Тени и эффекты: воспроизведите визуальные эффекты с помощью Flutter виджетов

## Проверка соответствия

- Сравнивайте каждый экран с соответствующим макетом
- Убедитесь, что компоненты выглядят одинаково на всех платформах
- Проверьте адаптивность на различных размерах экрана
