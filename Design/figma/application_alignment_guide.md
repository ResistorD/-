# Руководство по приведению приложения к эскизам Figma

## Обзор

Этот документ описывает шаги, необходимые для модификации текущего приложения Pressure Diary, чтобы оно соответствовало дизайну, представленному в эскизах Figma.

## Текущее состояние приложения

На момент анализа приложение имеет следующую структуру:
- Основные экраны: главный экран, экран истории записей, экран профиля
- Тема оформления: базовая светлая тема
- Компоненты: базовые Flutter виджеты
- Навигация: стандартная навигация между экранами

## Необходимые изменения

### 1. Обновление темы оформления

#### Цвета
1. Заменить текущую цветовую палитру на указанную в эскизах:
   - Первичный цвет: #3D8BFC
   - Фоновый цвет: #FFFFFF
   - Вторичный цвет: #F5F7FA
   - Цвет ошибок: #E53E3E
   - Цвет успеха: #38A169

2. Обновить файл `/lib/theme/` с новыми цветами и стилями

#### Типографика
1. Установить шрифты в соответствии с гайдами:
   - Заголовки экранов: 24sp, полужирный
   - Подзаголовки: 18sp, нормальный
   - Основной текст: 16sp, нормальный
   - Вспомогательный текст: 14sp, нормальный

### 2. Модификация экранов

#### Главный экран (`/lib/screens/home_screen.dart`)
1. Обновить дизайн кнопки записи давления:
   - Сделать закругленные углы 8dp
   - Увеличить высоту до 56dp
   - Применить первичный цвет (#3D8BFC)

2. Добавить компонент графика:
   - Разместить ниже кнопки записи
   - Настроить отображение последних измерений
   - Использовать линейный график

#### Экран записей (`/lib/screens/records_screen.dart`)
1. Добавить элементы фильтрации:
   - Компонент фильтра по диапазону дат
   - Разместить в верхней части экрана

2. Обновить стиль списка записей:
   - Высота элемента 80dp
   - Белый фон
   - Закругленные углы карточек

#### Экран профиля (`/lib/screens/profile_screen.dart`)
1. Добавить карточку информации пользователя:
   - Закругленные углы 12dp
   - Тень (elevation 2)
   - Размещение в верхней части экрана

2. Обновить список настроек:
   - Высота элемента 60dp
   - Белый фон

### 3. Обновление компонентов

#### Кнопки (`/lib/widgets/buttons/`)
1. Создать виджет PrimaryButton:
   - Закругленные углы 8dp
   - Высота 48dp
   - Первичный цвет фона

2. Создать виджет SecondaryButton:
   - Обводка вместо заливки
   - Прозрачный фон
   - Первичный цвет текста и обводки

#### Поля ввода (`/lib/widgets/input_fields/`)
1. Обновить стиль TextField:
   - Закругленные углы 8dp
   - Высота 56dp
   - Внутренние отступы 16dp по бокам
   - Подчеркивание при фокусе

#### Карточки (`/lib/widgets/cards/`)
1. Обновить стиль Card:
   - Закругленные углы 12dp
   - Тень (elevation 2)
   - Внутренние отступы 16dp

### 4. Адаптация под различные размеры экранов

1. Проверить корректное отображение на разных размерах
2. Обеспечить адаптивность компонентов
3. Тестировать как в портретной, так и в ландшафтной ориентациях
