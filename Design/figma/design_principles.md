# Принципы дизайна для приложения Pressure Diary

## Цель дизайна

Создать интуитивно понятный, удобный и эстетически приятный интерфейс для ведения дневника артериального давления, способствующий регулярному использованию приложения пользователями всех возрастов.

## Основные принципы

### 1. Простота использования
- Минимизация количества шагов для записи давления
- Четкая иерархия элементов
- Интуитивная навигация

### 2. Визуальная ясность
- Четкие заголовки и подписи
- Логическая группировка элементов
- Достаточный контраст текста и фона

### 3. Консистентность
- Единый стиль всех элементов
- Последовательное расположение компонентов
- Стандартные цвета и шрифты

## Цветовая палитра

### Основные цвета:
- **Первичный**: #3D8BFC (сине-голубой для акцентов и кнопок)
- **Фоновый**: #FFFFFF (чистый белый для основного фона)
- **Вторичный**: #F5F7FA (светло-серый для разделителей и второстепенных элементов)
- **Ошибки**: #E53E3E (красный для ошибок и предупреждений)
- **Успех**: #38A169 (зеленый для успешных действий)

### Цвета текста:
- **Основной**: #1A202C (темно-серый почти черный)
- **Вторичный**: #718096 (серый для подсказок)
- **Место для ввода**: #A0AEC0 (светло-серый для placeholder)

## Типографика

### Шрифты:
- **Основной шрифт**: Roboto или системный шрифт устройства
- **Заголовки**: Полужирный (weight 600-700)
- **Основной текст**: Обычный (weight 400)
- **Подсказки**: Тонкий (weight 300)

### Размеры текста:
- **Заголовки экранов**: 24sp
- **Подзаголовки**: 18sp
- **Основной текст**: 16sp
- **Вспомогательный текст**: 14sp
- **Мелкий текст**: 12sp

## Компоненты интерфейса

### Кнопки
- **Primary Button**: Закругленные углы 8dp, высота 48dp, основной цвет
- **Secondary Button**: Обводка, прозрачный фон, основной цвет текста
- **Icon Button**: Круглая форма, 48x48dp, иконка по центру

### Поля ввода
- Закругленные углы 8dp
- Высота 56dp
- Внутренние отступы 16dp по бокам
- Подчеркивание при фокусе

### Карточки
- Закругленные углы 12dp
- Тень Elevation 2
- Внутренние отступы 16dp

## Макеты экранов

### Главный экран
- Простой интерфейс для быстрой записи давления
- Большие кнопки для ввода значений
- График последних измерений

### Экран просмотра записей
- Хронологический список измерений
- Возможность фильтрации по датам
- Детали при нажатии на запись

### Экран профиля
- Личная информация пользователя
- Настройки уведомлений
- Параметры приложения

## Адаптивность

- Поддержка различных размеров экранов
- Корректное отображение на портретной и ландшафтной ориентациях
- Адаптация элементов под размер доступного пространства

## Доступность

- Поддержка увеличенного текста
- Достаточный размер сенсорных областей (>48dp)
- Соответствие стандартам WCAG
